# Bash-Projects

# Linux System Automation Scripts

This repository contains a set of Bash scripts focused on system administration automation, monitoring, and user management.

## Projects Included

### 1. System Health Monitoring
Monitors CPU, memory, disk usage, and service status with threshold-based logging.

 Directory: `system-monitor`

---

### 2. Log-Based Service Auto Recovery
Parses system logs to detect failures and performs controlled service recovery using systemd.

 Directory: `systemd_autohealing`

---

### 3. User Lifecycle Management
Automates user provisioning including group assignment, ACL configuration, and SSH key setup.

 Directory: `user_lifecycle`

---

## Skills Demonstrated

- Bash scripting and automation
- System monitoring and logging
- Log parsing using journalctl
- Service management with systemctl
- User and permission management (ACL)
- Secure SSH key deployment
- Idempotent script design

---

Each script is located in its respective directory with its own README file containing usage instructions.
